# Install
To install,
- Downlaod the mod [here](https://github.com/immcrab/GorillaCraftCompiled/releases/latest).
- Once downloaded, go to your Gorilla tag plugins folder (C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag\BepInEx\plugins).
- Launch Gorilla Tag, I personally recommend going into a modded lobby.
- Enjoy!
