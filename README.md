## Installation

Follow these steps to install GorillaCraft:

1. **Download the mod**  
   - Get the latest compiled version here: [Download Latest Release](https://github.com/immcrab/GorillaCraftCompiled/releases/latest)  

2. **Locate your Gorilla Tag plugins folder**  
   - Default path on Windows:  
     ```
     C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag\BepInEx\plugins
     ```  
   - If you installed Gorilla Tag elsewhere, navigate to that folder and then into `BepInEx\plugins`.

3. **Move the mod file**  
   - Copy the downloaded `.dll` file into the `plugins` folder.

4. **Launch Gorilla Tag**  
   - Start the game via Steam.  
   - For best results, join a [modded lobby](https://github.com/immcrab/Lasting-Utilla/releases/latest).

5. **Tips**  
   - Make sure **BepInEx** is installed and up-to-date.  
   - Some features may only work in modded lobbies.  
   - If the game crashes, check the log at `BepInEx\LogOutput.log`.
